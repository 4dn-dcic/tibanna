=========
Snakemake
=========

Tibanna supports Snakemake pipelines through the snakemake interface (``snakemake --tibanna``). Check out the Snakemake documentation_ for more details.

.. _documentation: https://snakemake.readthedocs.io/en/stable/executing/cluster-cloud.html#executing-a-snakemake-workflow-via-tibanna-on-amazon-web-services
